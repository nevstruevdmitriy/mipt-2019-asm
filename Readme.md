### Для kомпиляции, линковки и запуска выполнить
-> make

### Для oтладки выполнить
-> make debug

### Для yдаления сгенерированных файлов выполнить
-> make clean

### Для kомпиляции выполнить
-> make compil

### Для lинковки выполнить
-> make link

### Для zапуска выполнить
-> make run
